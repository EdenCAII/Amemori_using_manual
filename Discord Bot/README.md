# Discord Bot

#### Discord bot can realize AI robot painting, select or input some keywords to generate unique and exquisite paintings.

#### You can draw AI in two ways.

#### First method : select keywords for painting.

##### How to select keywords for painting?

###### Step 1 enter! button

#### Second method : input keywords for painting.

##### How to input keywords for painting?

###### Step 1 enter /image and add keywords

###### Note that you need to enter a space between the instruction /image and the keyword.

![72804d3c6705759f3025f2e025d27f1.png](https://github.com/EdenCAII/test_2/blob/main/72804d3c6705759f3025f2e025d27f1.png?raw=true)

###### Step 2 wait for AI robot to paint

###### Wait until the painting is loaded, and the painting progress will be displayed in real time.

![89083358a5c882ad16ecf119e6e2fef.png](https://github.com/EdenCAII/test_2/blob/main/89083358a5c882ad16ecf119e6e2fef.png?raw=true)
