# Discord Bot中文

#### discord Bot能够实现AI机器人作画，选择或者输入一些关键词生成独一无二的精美画作。

#### 你能够使用两种方式进行AI作画

#### 第一种方式：选择关键字进行作画

##### 如何选择关键字进行作画？

###### 第一步 enter!button

 ![1.png](https://github.com/EdenCAII/test_2/blob/main/1.png?raw=true)

###### 第二步 选择关键字

![2.png](https://github.com/EdenCAII/test_2/blob/main/2.png?raw=true)

###### 第三步 画作生成成功

![3.png](https://github.com/EdenCAII/test_2/blob/main/3.png?raw=true)

#### 第二种方式：输入关键字进行作画

##### 如何输入关键字进行作画？

###### 第一步 输入/image 加上关键字

###### 注意在指令 /image 与关键字之间需要输入空格分隔开。

![72804d3c6705759f3025f2e025d27f1.png](https://github.com/EdenCAII/test_2/blob/main/72804d3c6705759f3025f2e025d27f1.png?raw=true)

###### 第二步 等待AI机器人作画

###### 等待画作加载完成，作画进度会实时进行显示。

![89083358a5c882ad16ecf119e6e2fef.png](https://github.com/EdenCAII/test_2/blob/main/89083358a5c882ad16ecf119e6e2fef.png?raw=true)

![9637b3e32a75e957c5c2764bfc308f1.png](https://github.com/EdenCAII/test_2/blob/main/9637b3e32a75e957c5c2764bfc308f1.png?raw=true)

###### 当进程显示为100%时，画作生成成功。

###### 第三步 画作生成成功

###### 点击链接就能够下载使用关键字进行作画的原图。
