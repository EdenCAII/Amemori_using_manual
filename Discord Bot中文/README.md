# Discord Bot中文

#### discord Bot能够实现AI机器人作画，选择或者输入一些关键词生成独一无二的精美画作。

#### 你能够使用两种方式进行AI作画

#### 第一种方式：选择关键字进行作画

##### 如何选择关键字进行作画？

###### 第一步 输入!button

 ![1.png](https://github.com/EdenCAII/test_2/blob/main/1.png?raw=true)

###### 第二步 选择关键字

![2.png](https://github.com/EdenCAII/test_2/blob/main/2.png?raw=true)

###### 第三步 画作生成成功

![3.png](https://github.com/EdenCAII/test_2/blob/main/3.png?raw=true)

#### 第二种方式：输入关键字进行作画

##### 如何输入关键字进行作画？

###### 第一步 输入/image 加上关键字

###### 注意在指令 /image 与关键字之间需要输入空格分隔开。

![111.png](https://github.com/EdenCAII/test_2/blob/main/111.png?raw=true)

###### 第二步 等待AI机器人作画

###### 等待画作加载完成，作画进度会实时进行显示。

![222.png](https://github.com/EdenCAII/test_2/blob/main/222.png?raw=true)

![333.png](https://github.com/EdenCAII/test_2/blob/main/333.png?raw=true)

###### 当进程显示为100%时，画作生成成功。

###### 第三步 画作生成成功

###### 点击链接就能够下载使用关键字进行作画的原图。

![444.jpg](https://github.com/EdenCAII/test_2/blob/main/444.jpg?raw=true)

###### 快点进入Amemori，使用AI Art进行创作吧。
